# Demo Script

1. Run `.\start.ps1`.
2. Open `http://<your-machine-ip>:8000` from the demo machine or another device on the same LAN. Local access also works at `http://127.0.0.1:8000`.
3. Enter `https://github.com/pallets/flask`.
4. Wait until status is completed.
5. Show the repository profile.
6. Show the Tree-sitter CodeGraph: modules, symbols, import edges.
7. Open learning paths and compare 30-minute / 2-hour / 1-day routes.
8. Open contribution radar.
9. Generate a first-contribution mission.
10. Point to the Agent trace panel to explain why the system is not a black-box RAG chatbot.
